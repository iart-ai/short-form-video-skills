# TikTok Video Skills

> TikTok, Reels, and Shorts skills for AI coding agents — engineer short-form video that survives the swipe.

![TikTok Video Skills — animated showcase](./showcase.gif)

## Install

```bash
npx skills add iart-ai/tiktok-video-skills
```

Works with Claude Code, Cursor, Codex, GitHub Copilot, and 40+ agents.

## What's included

| Skill | What it does |
|---|---|
| [short-form-video](./short-form-video) | The hook → retention → loop grammar of vertical 9:16 video — hooks, pattern interrupts, pacing, and repeatable templates. |
| [caption-animation](./caption-animation) | Word-timed, karaoke-style captions from a transcript or voiceover — per-word pop, active-word highlight, safe-area placement. |
| [countdown-video](./countdown-video) | Frame-accurate, drift-free countdown timers with digit flip/roll motion and a configurable duration. |
| [lower-thirds](./lower-thirds) | Clean, branded name/role lower thirds with staged enter/exit animation — templated across a whole roster. |

## When it activates

- Making a Reels / TikTok / YouTube Shorts video, or building a reusable short-form template.
- Editing for retention: fixing the first 3 seconds, adding pattern interrupts, tightening pacing, making a seamless loop.
- Adding animated or karaoke captions, word-by-word subtitles, or burning captions onto a voiceover.
- Building a countdown timer — "starting soon" screens, launch/sale countdowns, animated counters that land on zero.
- Creating animated lower thirds, name tags, speaker overlays, or broadcast-style titles across a list of names.

## Example prompts

- "Make a 9:16 TikTok template and fix the first 3 seconds so it hooks."
- "Turn this voiceover transcript into Hormozi-style word-by-word captions."
- "Build a 5-minute 'starting soon' countdown that ends exactly on zero."
- "Add an animated lower third with the speaker's name and role."
- "Pace the cuts and add pattern interrupts so this Reel holds 70%+ retention."
